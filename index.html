Group Policy Management Policies

Powershell Scripts

Automation scripts for managing system policies in lab environments.

The following are the group policy object names as applied in the Active Directory.

Application Level

  • Domain Level Group Policies
    • Allow domain users to run powershell scripts
    • Default Domain Policy.
    • Desktop Wallpaper
    • GPO remote update windows firewall settings
    • LogData-Enquiry
    • Mamal-Talabaat
    • Mamal-Talebaat
    • Multimedia permission
    • Share_MapDrive
  • MAMAL ORGANIZATION UNIT LEVEL
    • ADD CHROME EXTENSIONS
    • GROUP-POLICY-Block Internet Websites in Microsoft Edge
    • Map Home Drive For Students
    • Restrict Incognito Browsing
  • TALABAAT & TALEBAAT ORGANIZATION LEVEL

    These are policies which are applied inside the Talebaat and Talabaat users

    • Block malicious downloads in browsers
    • Block List of websites which are not allowed in computer lab
    • Disable the CTRL+ALT+DELETE options
    • Folder Redirection
    • GPO TO INSTALL CHROME EXTENSIONS
    • Lab Client Deployment
    • Prevent Registry Editor
    • Prevent Users from installing chrome extensions

Computer Organization Unit Level

Controls applied at OU level for lab machines.

  • ADD CHROME EXTENSIONS
  • Allow Domain Computers to run powershell scripts
  • GPO-Restrict Chrome Extensions (Block *)
  • GPO-FOR DO NOT REQUIRE CTRL+ALT+DELETE
  • Prevent Temp Profiles
