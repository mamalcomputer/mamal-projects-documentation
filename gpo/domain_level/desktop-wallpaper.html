Desktop Wallpaper Group Policy

Purpose

This policy enforces a standard desktop wallpaper across all domain users to maintain branding consistency and prevent unauthorized changes.

How it works

The Group Policy Object applies a centralized wallpaper setting from the domain controller. When users log in, Windows retrieves and applies the configured image automatically.

How it is applied

  • Open Group Policy Management Console (GPMC)
  • Edit the target Domain Policy
  • Navigate to: User Configuration → Administrative Templates → Desktop → Desktop Wallpaper
  • Enable policy and set the image path
